Online Food Ordering System (Java Console Project)

Project Objective:
The Online Food Ordering System simulates a real-world digital ordering workflow used by restaurants and food delivery platforms. Users can browse menus, add food items to their cart, place orders, calculate billing, cancel orders, and view their order history. This project is built using Core Java with a clean, layered package structure and custom exception handling.

Features:
* Browse restaurant menu
* Add food items to cart
* Place orders with stock validation
* Auto-generate unique Order IDs
* Calculate total bill based on quantity
* Cancel orders with stock reversal
* View user order history
* Custom exceptions for robust error handling

Core Concepts Used:
* Object-Oriented Programming (OOP)
* Encapsulation with Getters & Setters
* Custom Exception Handling
* ArrayList Collections
* UUID for Order ID generation
* Layered architecture (Entity → Service → Main)
